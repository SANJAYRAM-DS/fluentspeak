# Frontend

This folder contains the standalone HTML, CSS, and JavaScript UI for FluentFlow AI.

Open `index.html` in a browser through a local static server, then point it at the Django backend running on `http://127.0.0.1:8000`.

Pages:
- `index.html`
- `dashboard.html`
- `chat.html`
- `vocabulary.html`
- `progress.html`

Assets:
- `assets/app.css`
- `assets/app.js`
